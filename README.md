## Run Locally

Clone the project

```bash
  git clone git@github.com:auddax/notes-keeper.git
```

Go to the project directory

```bash
  cd notes-keeper
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
